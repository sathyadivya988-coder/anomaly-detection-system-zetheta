# Market Anomaly Detection System

## Project Overview
This project builds a market anomaly detection platform to identify
fraudulent and manipulative trading activity.

## Objectives
- Detect unusual trading behavior
- Apply statistical and ML-based anomaly detection
- Reduce false positives
- Support investigation workflows

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- PyOD
- Jupyter Notebook

## Project Structure
- data/ : datasets
- notebooks/ : analysis notebooks
- src/ : source code
- docs/ : documentation
